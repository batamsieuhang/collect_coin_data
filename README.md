# Collect the most popular coin in the site
## **How to install**

**Requirement**
- python
- pip
- git

Command to install
```bash
git clone https://github.com/batamsieuhang/collect_coin_data.git
```

Command to use
```bash
cd collect_coin_data
```
```bash
pip install -r requirements.txt
```

```bash
python main.py
```

or

```bash
python3 main.py
```
