#### *Ví dụ về các chạy code*

**Ví dụ 1**

```bash
python main.py 1m 2m 3m 4m 2 5
```


- 1m 2m 3m 4m là các ô 1m, 2m, 3m, 4m
- 2 là tùy chọn hiển thị các % tăng của các coin xuất hiện từ 2 lần trở nên
- 5 là tùy chọn delay 5s sau mỗi lần chạy
